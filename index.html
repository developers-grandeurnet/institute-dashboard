A Dashboard system that simplifies all your institute Management

Key Features -

  • Add Branch, Branch Admin, Investors, Students, Employees, Parents, Enquiries, Batch, Courses.
  • Manage Branch, Branch Admin, Investors, Students, Employees, Parents, Enquiries, Batch, Courses.
  • Can see all important details on homepage of Dashboard like - count of students, branches, employees, investors.
  • Admins can create other dashboard users like front office, Library manager, Accounts department.
  • Library manager can manage all the books that are issued by any student and on which date.
  • Accounts department user can view and manage the fees of students and salaries of all employees.

Institute Admin Dashboard functionalities -

  • Can see the number of students, branches, Employees, Investors in homepage.
  • Can Add new Branches, Branch Admin, Investors.
  • Can Manage Branches, Branch Admins, Investors.
  • Manage Students, Search students based on their enrollment number, and batch.
  • Can Manage Employees i.e. teachers of institute. and send message to all of them seprately.
  • Has Login and Logout functionalities.

Branch Admin Dashboard functionalities -

  • Can Add new Students, Parents, Enquiries, Batches, Courses, Employees.
  • Can manage Students, Parents, Enquiries, Batches, Courses, Employees.
  • Create and manage user of Account Department Dashboard.
  • Create and manage user of Front Office Dashboard.
  • Create and manage user of Library Dashboard.
  • Can Schedule Test for students and See list of all tests and can delete them.
  • Has Login and Logout functionalities.

Account Depaprtment Dashboard functionalities -

  • Can see the total income and total profit of current month.
  • Can see the total payment of that day in Dashboard homepage.
  • Add payment and see payment history of students fees, can choose payment method between online and offline.
  • Can mark salaries and edit salaries of employee, can see history of salaries also.
  • See history and mark all the expenditures.
  • Also can view all students fee and mark them.
  • Has Login and Logout functionalities.

Front Office Dashboard functionalities -

  • Can Add Enquiries, Courses, Students, Employees, Batches and Parents.
  • Can Manage Enquiries, Courses, Students, Employees, Batches and Parents.
  • Adding a student, Employee and parent will send an email to the email, filled in the form.
  • Can Schedule Test for students and See list of all tests and can delete them.
  • Has Login and Logout functionalities.

Library Dashboard functionalities -

  • Can add book in list of registered books when any student borrow a book from library.
  • Can View and edit the details of a registered book.
  • Add detail when a student issue a book, with student Name and student id.
  • Can search students based on batch.
  • Has Login and Logout functionalities.
